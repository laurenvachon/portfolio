Private Club Theme

Web Design. Front-End Development.

Responsive | Bootstrap | SCSS | Flexbox | Public & Private

Launched April 2017

Worked with the largest third party manager of golf and Club operations in the world to create a theme that is now available to over 60 public & private clubs. This new theme is reflective of their new corporate branding that they’re launching, by using the same fonts and similar styles.

One of the goals of this theme was to make it as flexible as possible, as it could the design for over 60 private clubs. This was accomplished by using FlexBox so that the client could activate and deactivate pieces like callouts, videos, CTA’s, and having the site react accordingly.

Another goal was to also help the client show their branding. Creating a theme for over 60 private clubs and making each site feel unique and specific to each club was a challenge, but was tackled by using SCSS variables to create different shades using the clubs primary and secondary colors. The club has the flexibility to change out the colors in multiple areas of the site to maximize customization.
