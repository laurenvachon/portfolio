MX New England App Concept

UX & UI Design.

Sketch | Mobile App | Concept

Created March 2017

MX New England was created to help motocross riders in the New England area find events and tracks to ride. After designing their website, I decided to make the following concept app that explored the idea a bit further, by adding registration, event results, and further integration with your mobile device.

View Invision Prototype

Home screen that gives an overview of the most important information for racers: notifications, upcoming events & results.

Track list using Material design principles.

Notification screen, to be used for urgent notifications such as delays, cancellations, etc.

Results detail page that displays in a table.

Event detail page that shows all pertinent information for the event.

Has the ability to allow motocross riders to register for events after reading all of the event details.
